## 🚀 Como Executar Localmente

### Pré-requisitos
- Node.js v18+
- npm v9+

### Instalação
```bash
# Clone o repositório
git clone https://github.com/Benevanio/portfolio.git

# Instale as dependências
npm install

# Execute em modo desenvolvimento (com hot reload)
npm run dev

# Ou para produção
npm start
```
